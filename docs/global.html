fitToMarkers()

Changes bounds of map to view all markers.

Note: after calling this function, this.bounds, this.center, and this.zoom may temporarily be null as the map moves. Therefore, use this.addMapListenerOnce if you need to access these values immediately after calling fitToMarkers.
