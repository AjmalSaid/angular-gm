Module: AngularGM

Module for embedding Google Maps into AngularJS applications.

API Documentation

See...

  • module:gmMap for usage of the <gm-map> directive
  • module:gmMarkers for usage of the <gm-markers> directive
  • module:gmInfoWindow for usage of the <div gm-info-window="...> directive
  • module:angulargmContainer if you need to run custom configuration on the map, e.g. add new map types
  • module:angulargmDefaults to override the default map options
